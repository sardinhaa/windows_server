Active Directory: 
(autenticaçao/autorizaçao)

autenticaçao: 
		resposabiliza se por fazer com que um utelizador aceder os recursos da organizaçao, ou seja garantir que este tenha acesso a rede atrasves do id, este e unico a palavra passe e user.

autorizaçao:
		determina o que este user pode o nao aceder ou seja criar restriçoes e os previlegios 

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Domain Controller:

	ajuda a gerenciar os acessos ao servidor, respodendo a pedidos de autenticaçao 

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Domínio:
	
	um dominio é uma base de dados central, no servidor, se encontram os objetos: utilizadores, computadores, impressoras, etc  
	
--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- 
dominio vs wkgroup:

    dominio: centralizado, usado para redes grandes;
    
    wkgroup: descentralizado, usado para redes pequenas;

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
gpos (group policy, politicas de grupo):

     servem para gerenciar os utilisadores e os seus ambientes;
